APPMARX enables Bloggers to network with like minded Bloggers.

== Description ==

The widget connect your Blog with other bloggers in a visually compelling way.  You form connections just like in a Social Network (on appmarx.com/ ).  See: http://wordpress.org/extend/plugins/appmarx-blog-networking/screenshots/ .

== Installation ==

1. Upload plugin zip file to the `/wp-content/plugins/` directory.
2. Activate the plugin via the 'Plugins' menu in WordPress.
3. Go to http://appmarx.com/, register and collect snippet code.
4. On your WP dashboard, go to "AppMarx Widget" under "Appearance" and insert your code: http://appmarx.com/Loaders/Loader_######.js .
5. Replace ###### with your APPMARX code.
